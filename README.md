# HR_Viz
This project will involve simple ETL processing, visualization and business reporting dashboards using Python, R and Tableau SQL.

This project uses https://www.kaggle.com/datasets/rhuebner/human-resources-data-set?resource=download as a sample dataset.

This file is then processed into a business-ready format via simple ETL processing using Jupyter Notebook.

Next steps would be to visually animate it using R and publish this on RPubs. The following step is to create a Tableau dashboard to make it interactive, drilldown-embedded, metric heavy and business reporting worthy. This would be then published on Tableau Public (https://public.tableau.com/app/profile/aarushi.k7283/vizzes).
